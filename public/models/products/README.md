# Product 3D Models

Drop product-specific `.glb` or `.gltf` files in this folder.

For the demo, only these two products are configured to show the 3D icon:

- `/models/products/prod_terra_grip.glb`
- `/models/products/prod_trailform_jacket.glb`

If a configured product file is missing, the viewer falls back to the generated Three.js model.
